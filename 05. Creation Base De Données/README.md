# Creation BDD

+ Créer un schéma MySQL : `BookstoreDB`

+ Créer 7 tableaux :

    + Users
    + Category
    + Book
    + Customer
    + Review
    + BookOrder
    + OrderDetail


# Créer un schéma `BOOKSTOREDB`

+ Ouvrir le client de ligne de commande MySQL
+ Connectez-vous au serveur
+ Tapez :

    ```
    create database `bookstoredb`
    ```

+ Utilisez quelques commandes :

    ```
    show databases
    connect <schema>
    show tables
    ```
